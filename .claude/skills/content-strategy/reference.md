# Content Strategy Reference

## Platform Formats

### LinkedIn
- **Length**: 150-300 words (1300 char sweet spot)
- **Format**: Short paragraphs (1-2 sentences each), line breaks between
- **Hook**: First 2 lines visible before "see more" — make them count
- **Hashtags**: 3-5 relevant, at bottom
- **Best time**: Tue-Thu 8-10am
- **Tone**: Professional, authoritative, approachable

### Twitter/X
- **Length**: 240 chars per tweet, threads for depth
- **Format**: Thread = hook tweet + 3-7 value tweets + CTA tweet
- **Hook**: Provocative statement or surprising stat
- **Best time**: Mon-Fri 9am-12pm
- **Tone**: Sharp, concise, personality-driven

### Blog Post
- **Length**: 800-1500 words
- **Structure**: Problem > Solution > Implementation > CTA
- **SEO**: Primary keyword in title, H1, first paragraph, meta description
- **Code**: Include code snippets where relevant
- **Headers**: H2 for sections, H3 for subsections

### Email
- **Subject**: < 50 chars, curiosity-driven
- **Preview text**: Complements subject, doesn't repeat it
- **Body**: Single CTA, mobile-first formatting
- **Length**: 150-300 words

### YouTube
- **Title**: < 60 chars, keyword + curiosity gap
- **Description**: First 150 chars visible, include links and timestamps
- **Script**: Hook (30s) > Intro (30s) > Content (5-15min) > CTA (30s)

## Content Pillars (AI Skills Brand)
1. **Expertise** (40%) — Deep AI, software, automation knowledge
2. **Results** (25%) — Case studies, metrics, before/after
3. **Insights** (20%) — Industry trends, predictions, analysis
4. **Culture** (15%) — Team, values, behind-the-scenes

## Weekly Rhythm
- Monday: Industry insight / trend analysis
- Tuesday: How-to / tutorial
- Wednesday: Case study / results
- Thursday: Thought leadership / opinion
- Friday: Community / engagement / lighter content

## Words to Avoid
- Very, really, basically, actually, honestly
- "AI is revolutionizing/transforming..."
- "In today's fast-paced world..."
- "Leverage", "synergy", "paradigm shift"
- Anything that sounds like AI wrote it without editing
